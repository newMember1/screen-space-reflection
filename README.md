# screen-space-reflection
ssr demo
利用g-buffer实现一个简易的线追踪demo。

效果如图：
![image](https://github.com/newMember1/screen-space-reflection/blob/master/pics/Screenshot%20from%202021-09-06%2023-22-43.png)

使用方法(linux)：
```
mkdir build
cd build
cmake ..
make
./screen-space-reflection
```
